Site

Functions for WordPress site.

Login

Login to WordPress site.

wp_login(email, password)

Type

  • :type email: str
  • :type password: str

Description

  • :param email: email or username of WordPress User
  • :param password: password of WordPress User

Example

wp.wp_login('admin@mysite.com', 'password')
